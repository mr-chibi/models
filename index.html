Welcome to Mr_Chibi's Models Wiki!

You'll find much information about Recent Updates, Commands, and Guides about how to create your own models using this datapack.

    What is Mr_Chibi's Model Datapack?

    Mr_Chibi's Models Datapack, is a program that allows players to Create Block Models and Effects!!

    What is Mr_Chibi's Model Datapack?

    The world is the players pallete, either it be for decoration, design, or experimenting.
    Models can be made in "Singleplayer" or "Multiplayer". They can be exported over using the games in game mechanics for saving inventory slots and imported into a server. As long the server is running models program anything that is made from singleplayer or another server can be loaded in!

    How does models render?

    Every block model that is loaded into the game is a "minecraft:block_display". When Players, save their models they've created all the data gets sanitized into my own custom nbt. An saved into a custom spawn egg with the data. So servers don't struggle by default minecrafts code and eat tps. I want to ensure models works for everyone so the fun can be spread across other "servers", "realms" or "singleplayer"!

    What Features does it offer?

    Program provides many tools such as,

    • Texturing
    • Grouping Palletes
    • Rotations
    • Angles
    • Scaling
    • Axis Adjusting
    • Aligning
    • Copy
    • Pasting
    • Animations
    • Import/Exporting
