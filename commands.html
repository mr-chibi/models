Models Player Commands

You'll find information here about how to use commands and small descriptions about how they functions and what they do.

Getting Started

Adding Permissions

How do I enable Admin Commands?

  • If you're a Server Operator you can grant yourself or another player permissions by doing the following command, "/team join model_admin [username]"

Removing Permissions

  • If you're a Server Operator you can also remove the permissions from the player, by doing the following command, "/team leave [username]"

Public Commands

Commands

Understanding the code

Commands regular players, can use in survival to use brushes, texture, & remove.

  • /trigger model_tools — Grants Player's list of tools for texturing, removing, palleting, etc.
  • /trigger model_paste_rotation — Toggling it once will [Enable] feature to paste saved rotated models, but toggling again will [Disable] feature.

Brushes

How these types of brushes work!

Types of brushes with descriptions.

  • Texture Brush — Edits indivdual textures when using /trigger forge_edit_layer set [layer] command.
  • Primary Brush — Edits all textures of primary layer on the model.
  • Secondary Brush — Edits all textures of secondary layer on the model.
  • Triad Brush — Edits all textures of triad layer on the model.
  • Eraser Brush — Right clicking will remove the model and tell you in chat it has been removed can also remove collison blocks such as, barriers, oak fence, etc.
  • Paste Brush — When using the command, /trigger model_copy it will copy nearest model and when right clicking with brush it will render and paste it in.
  • Sel Brush — Stops all current selections from editing or creating.

Removing Models

How do I remove models in game?

For Removing models in game.

  1. First, /trigger model_tools — gives player brushes.
  2. Second, In your inventory look for the Eraser Brush place it in your Main hand.
  3. Lastly, Right click with the brush at the model until it disappears tell you in chat. Ensure if it has collision to continue right clicking till barriers are removed.
  4. Once, the Model has been removed you'll be given a spawn egg back with the data.

Admin Commands

Commands

What do these commands do?

  • /trigger model_sel — Stops current selection for what player is editing.
  • /trigger model_copy — Will copy closest model Player is near by and give player a new paste brush.
  • /trigger model_paste — Toggles automaticly weither player has "right clicked" or "thrown minecraft:allay_spawn_egg" on the floor.
  • /trigger model_rotation — min value of 1 and max value of 360. Rotates entire model.
  • /trigger model_ui — Text box GUI appears players can interact with it and get custom model eggs!

Forge Mode

Commands

What do these commands do?

  • /trigger forge_tools — Grants Model Admin tools for easier forge creation without typing commands.
  • /trigger forge_speed — Speeds the objects [Movement or Rotation] using forge tools. Players, can set it to any value.
